## üõ†Ô∏è Full-Stack Setup (Frontend + Backend)

To enable the contact form and backend features:

### Prerequisites
- [Node.js](https://nodejs.org/) installed.
- [MongoDB](https://www.mongodb.com/try/download/community) installed and running.

### Installation

1.  Open a terminal in this folder.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the server:
    ```bash
    npm start
    ```
4.  Open `http://localhost:3000` (or the port shown in the terminal) in your browser.

## ‚ùì Troubleshooting

### Error: "'npm' is not recognized..."
This means **Node.js** is not installed on your computer.

**Solution:**
1.  Download Node.js from [nodejs.org](https://nodejs.org/).
2.  Install it (keep all default settings).
3.  **Restart your computer** (or close and reopen VS Code) for the changes to take effect.
4.  Try running `npm install` again.

### Can I view the site without Node.js?
**Yes!** You can view the entire design without installing anything.
- Just double-click **`index.html`** in the `google antigrivity` folder.
- **Note:** The "Send Message" button on the Contact page won't send emails until you install Node.js and start the server.
